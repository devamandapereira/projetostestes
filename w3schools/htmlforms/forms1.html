forms

os forms html sao usados pra coletar o input do user. o input na maioria das vezes é enviado para um serve pra ser processado.

é preciso adicionar um label para o que é desejado adicionar e o título dentro do label, e aí sim adiciona-se o input.

o "action" do frame é pra quando houver input, assim o valor do input será enviada para outro local.

há outras especificações e particularedades do input, mas é necessario haver conhecimento em PHP.

nesse documento há a base necessária para criar, pelo menos, uma área de cadastro de rascunho.
